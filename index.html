Short biography
Raphael Carvalho is a computer programmer who loves open source, and especially kernel programming. Raphael made his first steps into the open source world, when he sent a patch to slightly improve the PID allocation code from the Linux kernel. He has worked on bringing new file system support and additional features for the open source project Syslinux, which is a suite of lightweight IBM PC bootloaders. He works at Cloudius Systems, an israeli startup. Nowadays, he's mostly involved with the file system technology from OSv, a cloud-based operating system.
